# 📄 Smart Attendance System (Frontend)

## 📌 Project Title
**Smart Attendance System using HTML, CSS, and JavaScript**

---

## 📖 Description
This project is a **Smart Attendance System (Frontend Simulation)** developed using HTML, CSS, and JavaScript.

It allows users to mark attendance using:
- 📷 Face Attendance (simulated)
- 🆔 ID-based Attendance

All attendance data is stored in the browser using **localStorage**, so records remain even after refreshing the page.

---

## 🎯 Features
- 🔐 Login Page (UI only)
- 📊 Dashboard Interface
- 📷 Face Attendance Simulation
- 🆔 ID-based Attendance Entry
- 📅 Real-time Date & Time Display
- 📋 Attendance Records Table
- 💾 Data stored in localStorage
- 🔄 Auto-load records after refresh
- 🧹 Clear Attendance Records
- 🚪 Logout Function
- 📱 Responsive Design (Mobile + Desktop)

## 🧪 How It Works

### 🔐 Login
- Enter any username and password
- Click login → dashboard opens

### 📷 Face Attendance
- Click "Capture Face"
- Adds a record:
  - Type: Face
  - Name: Face User
  - Time: Current time

### 🆔 ID Attendance
- Enter Student ID
- Click Submit
- Record is added with time

### 📋 Records Table
- Displays all attendance entries
- Data updates instantly

### 💾 Storage
- Data is saved in **localStorage**
- Records remain after page refresh

### 🧹 Clear Data
- Click "Clear Attendance"
- Removes all stored records
